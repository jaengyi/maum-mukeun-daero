# 02. 제품 요구사항 명세서 (PRD)

> Document type: Product Requirements Document
> Scope: MVP 0.1.0 (턱걸이 10개 달성 시나리오)

---

## 1. 제품 정의

### 1.1 무엇을 만드는가
**마음먹은대로**는 사용자가 입력한 신체 정보 및 현재 능력을 기반으로 **개인화된 트레이닝 계획을 시뮬레이션**하고, 매일의 수행을 트래킹하며, **시각적 성장 보상(잔디)**으로 동기를 유지시키는 안드로이드 네이티브 앱이다.

### 1.2 MVP 범위
- 단일 카테고리: **운동 / 턱걸이 10개 달성**
- 단일 사용자: 회원가입 없이 로컬 데이터 우선 (선택적 클라우드 백업은 Phase 1+)
- 단일 플랫폼: Android (최소 SDK 26 / Android 8.0)

### 1.3 MVP 범위 외 (Out of Scope)
- 다른 운동 종목 (러닝, 스쿼트, 푸시업 등)
- 학습 / 독서 / 식단 카테고리
- 소셜 기능 (친구, 랭킹, 챌린지)
- 코치 마켓플레이스
- iOS / Web

## 2. 사용자 페르소나

### 2.1 Primary Persona

**김작심 (32, 직장인 남성)**
- 키 175cm, 몸무게 80kg
- 운동 경력 거의 없음, 현재 턱걸이 1~2개 가능
- 헬스장은 부담스럽고 집/공원 철봉이 가까움
- 핸드폰: Galaxy S23
- 동기: "최소한 턱걸이 10개는 할 수 있는 사람이 되고 싶다"
- 좌절 포인트: "어떻게 늘려야 할지 모르겠고, 이대로가 맞는지 막막함"

### 2.2 사용자 여정 (User Journey)

1. **발견** → "턱걸이 늘리는 앱" 검색 / 지인 추천
2. **첫 실행 (Onboarding)** → 키, 몸무게, 현재 턱걸이 가능 횟수 입력
3. **시뮬레이션 결과 확인** → "당신은 약 10주 후 턱걸이 10개 달성 가능합니다" + 주차별 계획 미리보기
4. **첫날 Task 수행** → 오늘 할 운동 카드 등장 → 수행 후 기록
5. **잔디 첫 칸 채워짐** → 즉각적 보상 경험
6. **Day 7~14** → 패턴화, 알림으로 리마인드, 잔디 줄 길어짐
7. **Day 30+** → 정체 / 컨디션 변화 → 자동 재조정 알림
8. **Day 70~84** → 목표 달성, 새 목표 설정 또는 유지 모드

## 4. 비기능 요구사항 (Non-Functional)

### 4.1 성능
- 앱 콜드 스타트 2초 이내 (중급 안드로이드 기준)
- 모든 화면 전환 60fps 유지
- DB 조회 50ms 이내

### 4.2 데이터 / 저장소
- 로컬 우선 (Room DB)
- 사용자 데이터는 디바이스 외부 전송 없음 (MVP)
- 백업은 JSON 파일 내보내기 (사용자가 수동 저장)

### 4.3 접근성
- TalkBack 지원 (모든 액션에 contentDescription)
- 폰트 크기 시스템 따름
- 색맹 대응 (잔디 색상은 색뿐 아니라 명도 차이로도 구분 가능)

### 4.4 호환성
- Android 8.0 (API 26) 이상
- 화면 크기 6.0~6.8인치 우선 검증, 폴더블 대응은 Phase 1+

### 4.5 보안 / 개인정보
- 헬스 데이터(키, 몸무게)는 로컬 DB 암호화 저장 (SQLCipher 검토)
- 어떤 데이터도 외부 서버로 자동 전송하지 않음
- 분석 / 크래시 리포트는 사용자 동의 후에만

### 4.6 신뢰성
- 크래시율 0.5% 이하
- 운동 기록은 절대 손실되지 않음 (write-ahead-log, 즉시 fsync)

## 6. 사용자 스토리 핵심 5선

### US-1
> **사용자로서**, 첫 실행 시 내 신체 정보와 현재 능력을 입력하면 **앱이 자동으로 10개까지의 경로를 시뮬레이션**해서 보여주기를 원한다.
> **그래야** 어디서부터 어떻게 해야 할지 막막하지 않다.

**완료 조건**: 입력 완료 후 5초 이내에 주차별 계획 시각화가 표시된다.

### US-2
> **사용자로서**, 매일 홈 화면을 열면 **오늘 해야 할 단 하나의 미션**이 카드로 떠 있기를 원한다.
> **그래야** 결정 피로 없이 바로 시작할 수 있다.

**완료 조건**: 홈 화면 진입 시 첫 화면에 "오늘의 미션" 카드가 보이고, 한 번 탭으로 운동 시작 화면으로 진입한다.

### US-3
> **사용자로서**, 미션을 끝내면 **잔디 칸이 즉시 채워지는 애니메이션**을 보고 싶다.
> **그래야** 즉각적 보상으로 다음 날도 하고 싶어진다.

**완료 조건**: 마지막 세트 완료 시 0.5초 이내에 잔디 칸이 채워지는 모션이 시작되며 햅틱 피드백이 동반된다.

### US-4
> **사용자로서**, 정체기에 빠지면 **앱이 알아서 강도를 낮추고 회복일을 추가**해주기를 원한다.
> **그래야** 부상이나 좌절로 그만두지 않는다.

**완료 조건**: 최근 7일간 수행률 50% 미만 + 컨디션 평균 3 이하면 자동으로 재조정 카드 생성.

### US-5
> **사용자로서**, **누적 잔디**를 한눈에 보고 "내가 이만큼 했구나"를 느끼고 싶다.
> **그래야** 그만두기 아까워서 계속한다.

**완료 조건**: 성장 화면에서 12주 잔디 + 누적 운동일 수 + 누적 풀업 총 횟수가 한 화면에 표시된다.
